# NextJS를 활용한 영화정보 웹페이지

- NextJS 공부용도로 개발한 영화정보 웹페이지 입니다.

- 실행 방법

```
git clone https://github.com/kylew1004/next-js_study.git
```

```
cd nextjs-intro
```

```
npm i next react react-dom
```

```
// .env 파일을 생성하여 API_KEY = ${themoviedb.org 에서 발급받은 API_KEY} 를 넣어주세요!
npm run dev
```
